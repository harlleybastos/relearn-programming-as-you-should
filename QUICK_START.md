# 🚀 CodeQuest Quick Start Guide

## 📋 Copy-Paste Prompts for Every Situation

### 🎮 First Time Ever
```
Initialize me as a new CodeQuest student. My name is [YOUR_NAME].
Start me on the Tutorial Island to learn the system.
```

### 🔄 Starting Any New Chat Session
```
Continue my CodeQuest journey. 
Load my progress from @.context/current-session.md
Show me my current status dashboard.
```

### 💾 Manual Save
```
/checkpoint
Create a checkpoint with my current progress and all context.
```

### 🚨 When Context Limit Is Near
```
CONTEXT LIMIT WARNING - Execute full preservation:
1. Create comprehensive checkpoint
2. Generate knowledge crystal  
3. Give me exact continuation instructions
4. Save all active code and progress
```

### 🎯 After Creating New Chat (Following Context Limit)
```
Continue CodeQuest from crystal: [CRYSTAL_ID]
Restore my full progress and show status.
Resume from: [LAST_ACTIVITY]
```

### 📊 Check Your Progress
```
Show me my CodeQuest dashboard with:
- Current level and XP
- Active quest progress
- Available power-ups
- Recent achievements
```

### 💡 When Stuck
```
I'm stuck on [PROBLEM]. 
Can I use a hint token? (Current tokens: X)
Guide me without giving the solution.
```

### 🏆 View Achievements
```
Show me all my achievements and progress towards locked ones.
What's my next achievement target?
```

## 🎯 Quick Tips

1. **ALWAYS start new chats** with the continuation prompt
2. **Watch for "summarizing"** = context limit approaching
3. **Save before closing** Cursor with `/checkpoint`
4. **One quest at a time** for best learning
5. **Type everything yourself** - no copy-paste!

## ⚡ Emergency Recovery

If something goes wrong:
```
/codequest-recovery
Search all checkpoints in @.context/checkpoints/
Load the most recent one
Show me what was recovered
```

## 📱 Session Status Check
```
/session-info
Show:
- Current session ID
- Time played this session
- XP earned this session
- Challenges completed
- Last checkpoint time
```

---
**Remember**: The system auto-saves every 5 challenges. Just focus on learning!
